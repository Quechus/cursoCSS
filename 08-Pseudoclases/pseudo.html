Pseudoclases

Estilos especiales dependiendo del contexto,posición o estado.

:root

La pseudoclase de CSS ":root" selecciona el elemento raíz de un documento aplicado a HTML; :root representa el elemento <html> y es identico al selector html, excepto que su nivel de especificidad es mayor. **Sirve para declarar variables CSS**

:hover

La pseudoclase de CSS ":hover" se presenta cuando el usuario coloca el puntero sobre algun elemento, pero no necesariamente está activo.En pantallas touch :hover es problematico.

:active

La pseudoclase ":active" de CSS cubre el caso en el que un elemento está siendo activado por el usuario.Cuando se interactua con el ratón, se corresponde al momento en el que el usuario presiona el botón del ratón hasta el momento en que lo suelta.A menudo es usado en los elementos <a>y <button>de HTML.

El orden si importa:

  1. :link{} ---->relacionado a hipertextos
  2. :visited{} ---->relacionado a hipertextos visitados
  3. :hover{}
  4. :active{}

:target

La pseudoclase ":target" representa el elemento único, si existe alguno, con un id coincidente con el identificador de fragmentos de la URI del documento (Permite que la pagina sea responsiva sin necesidad de Javascript)

Cerrar

:nth-child :first-child :nth-last-child

  • nth-child:selecciona el elemento indicado que sea el enésimo hijo de su elemento PADRE.
  • first-child:selecciona el elemento indicado que sea el primer hijo de su elemento PADRE.
  • nth-last-child:selecciona el elementoindicado que sea el ultimo hijo de su elemento PADRE.

:nth-of-type :first-of-type :nth-last-of-type

  • nth-of-type:selecciona el elemento indicado que sea el enésimo hijo en su TIPO sin importar la jerarquia dentro del elemento padre.
  • first-of-type:selecciona el elemento indicado que sea el primer hijo en su TIPO sin importar la jerarquia dentro del elemento padre.
  • :nth-last-of-type:selecciona el elemento indicado que sea el ultimo hijo en su TIPO sin importar la jerarquia dentro del elemento padre.

Ejemplo interactivo

:not

La pseudoclase de CSS de negacion ":not(x)", es una notación funcional que toma un selector simple X como argumento. Coincide con un elemento que no está representado por el argumento. X no debe contener otro selector de negacion.

:empty

La pseudoclase ":empty" corresponde a un elemento sin ningun nodo hijo (vacio). Solo cuando hay nodos de elemento u texto con uno o más caracteres dentro del elemento ése se convierte en no vacio.
